About This Project

This project is an Insurance Charges Predictor that uses Machine Learning to estimate the medical insurance charges for individuals based on demographic and health-related parameters.

It considers various features such as age, gender, BMI, number of children, smoking status, and region to generate accurate charge predictions.

The model is trained on real-world insurance datasets and helps individuals or insurance agents assess risk, plan premiums, and understand how different factors influence charges.

Built using Flask, Python, and Bootstrap, this app provides a user-friendly interface where users can enter personal information and receive real-time cost predictions.
